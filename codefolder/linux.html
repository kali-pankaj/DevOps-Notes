2. Input / Output Redirection

CommandDescription
set -o noclobberPrevent overwriting files
set +o noclobberAllow overwriting files
ls > file.txtOverwrite output to file
ls >> file.txtAppend output to file
ls > output.txt 2>&1Redirect output & error to one file
ls >/dev/null 2>&1Silent execution (no output)
> file.txtClear file contents

6. Shell Operators & Symbols

SymbolDescription
;Execute multiple commands sequentially
&Run command in background
?Stores exit status of last command ($?)
&&Execute next command only if previous succeeds
||Execute next command only if previous fails
#Comment line in shell script
\Escape special characters
' 'Single quotes: treat everything literally
" "Double quotes: allow variables ($) expansion
|Pipe output of one command to another
